# ALI - Futuristic AI Assistant

A production-ready web application with voice, real-time search, and long-term memory.

## Features

- Natural Voice Conversations with ElevenLabs
- Real-Time Internet Search via SearchAPI.io
- Long-Term Memory with Supabase
- Streaming Responses with Gemini 2.5 Flash
- Cinematic Sci-Fi UI

## Tech Stack

- Next.js 15, React 19, TypeScript, Tailwind CSS
- Supabase PostgreSQL
- Google Gemini API
- ElevenLabs for voice
- SearchAPI.io for web search

## Getting Started

```bash
npm install
cp .env.example .env.local
npm run dev
```

Open http://localhost:3000
